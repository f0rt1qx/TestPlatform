# 📤 Импорт тестов из CSV

Система импорта тестов позволяет быстро загружать вопросы и ответы из CSV файлов.

## 📋 Формат CSV файла

Файл должен содержать следующие колонки:

| Колонка | Обязательная | Описание |
|---------|-------------|----------|
| `test_title` | ✅ | Название теста |
| `test_description` | ❌ | Описание теста |
| `time_limit` | ❌ | Лимит времени в минутах (по умолчанию 30) |
| `max_attempts` | ❌ | Максимум попыток (по умолчанию 1) |
| `pass_score` | ❌ | Проходной балл в % (по умолчанию 60) |
| `question_text` | ✅ | Текст вопроса |
| `question_type` | ❌ | Тип: `single`, `multiple`, `text` (по умолчанию `single`) |
| `points` | ❌ | Баллы за вопрос (по умолчанию 1) |
| `answer_text` | ✅ | Текст варианта ответа |
| `is_correct` | ✅ | `1` для правильного ответа, `0` для неправильного |

## 📝 Примеры

### Вопрос с одним правильным ответом (single)

```csv
test_title,test_description,time_limit,max_attempts,pass_score,question_text,question_type,points,answer_text,is_correct
"Математика","Базовый тест",30,2,60,"Сколько будет 2+2?","single",1,"4",1
"Математика","Базовый тест",30,2,60,"Сколько будет 2+2?","single",1,"5",0
"Математика","Базовый тест",30,2,60,"Сколько будет 2+2?","single",1,"6",0
```

### Вопрос с несколькими правильными ответами (multiple)

```csv
test_title,test_description,time_limit,max_attempts,pass_score,question_text,question_type,points,answer_text,is_correct
"Математика","Базовый тест",30,2,60,"Какие числа чётные?","multiple",2,"2",1
"Математика","Базовый тест",30,2,60,"Какие числа чётные?","multiple",2,"4",1
"Математика","Базовый тест",30,2,60,"Какие числа чётные?","multiple",2,"3",0
"Математика","Базовый тест",30,2,60,"Какие числа чётные?","multiple",2,"5",0
```

## 🚀 Как использовать

1. Откройте **Админ-панель** → вкладка **Тесты**
2. Нажмите **📤 Импорт CSV**
3. Выберите CSV файл
4. Нажмите **Импортировать**

## 💡 Советы

- **Кодировка**: Сохраняйте файл в UTF-8 для поддержки кириллицы
- **Группировка**: Все строки одного теста должны идти подряд
- **Повторный импорт**: Если тест с таким названием существует, вопросы будут добавлены к нему
- **Минимум ответов**: Каждый вопрос должен иметь хотя бы один вариант ответа

## ⚠️ Возможные ошибки

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `Пустое название теста` | Колонка `test_title` пустая | Заполните название для всех строк |
| `Пустой текст вопроса` | Колонка `question_text` пустая | Добавьте текст вопроса |
| `Пустой текст ответа` | Колонка `answer_text` пустая | Добавьте текст ответа |
| `Отсутствует колонка: X` | Нет обязательной колонки | Проверьте заголовки CSV |
| `Только CSV файлы` | Неверный формат файла | Сохраните файл как .csv |

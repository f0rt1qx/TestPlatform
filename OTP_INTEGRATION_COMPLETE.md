## 🎯 Как работает:

### Обычный вход (с паролем):
```
login.php → Email + Пароль → JWT → Dashboard
```

### OTP вход (без пароля):
```
login.php → "Вход по коду" → login-otp.php
  или сразу: http://localhost:8080/test-platform/login-otp.php

Шаг 1: Ввести email
Шаг 2: Получить код на email (6 цифр)
Шаг 3: Ввести код в 6 ячеек
Шаг 4: ✅ Вход выполнен!
```

# PDF Экспорт результатов

## Обзор

Реализована функция экспорта результатов тестирования в формате PDF для двух ролей:
- **Администратор** — выгрузка всех результатов
- **Студент** — выгрузка отдельного результата теста

## Файлы реализации

### 1. `src/helpers/PDFExporter.php`
Класс для генерации PDF документов через HTML-to-PDF конвертацию.

**Основные методы:**
- `exportResults(array $results)` — экспорт всех результатов (админ)
- `exportSingleResult(array $result, array $questions)` — экспорт одного результата (студент)

**Возможности:**
- Поддержка UTF-8 (кириллица)
- Красивое форматирование с градиентами
- Цветовая индикация статусов
- Статистика и визуализация данных
- Детализация ответов с вопросами

### 2. `api/admin.php`
**Endpoint:** `GET /api/admin.php?action=export_pdf`

Экспортирует все результаты тестирования.

**Доступ:** Только администраторы

### 3. `api/test.php`
**Endpoint:** `GET /api/test.php?action=export_pdf&attempt_id={id}`

Экспортирует результат конкретной попытки.

**Доступ:** Авторизованные пользователи (только свои результаты)

**Безопасность:**
- Проверка принадлежности результата пользователю
- Валидация attempt_id
- Авторизация через JWT middleware

### 4. `admin.php`
Кнопка экспорта во вкладке "Результаты":
```
📄 Экспорт PDF
```

### 5. `dashboard.php`
Кнопка экспорта в таблице истории:
```
📄 PDF (в колонке "Действия")
```

### 6. `test.php`
Кнопка на экране результатов после прохождения теста:
```
📄 Скачать PDF
```

## Структура PDF документа

### Админ-панель (все результаты)

1. **Шапка**
   - Название платформы
   - Дата формирования
   - Общее количество результатов

2. **Сводная статистика**
   - Всего тестов / Сдано / Не сдано
   - Средний балл
   - Средний уровень честности

3. **Таблица результатов**
   - Пользователь (имя + email)
   - Название теста
   - Попытка #
   - Процент
   - Статус (Сдан/Не сдан)
   - Уровень честности
   - Дата

### Студент (отдельный результат)

1. **Шапка**
   - Название платформы
   - Дата и время формирования

2. **Бейдж результата**
   - Статус (ТЕСТ СДАН / ТЕСТ НЕ СДАН)
   - Процент (большой шрифт)
   - Название теста

3. **Информационная сетка**
   - Пользователь (имя + email)
   - Тест (название + попытка)
   - Балл (получено/максимум + проходной)
   - Время (минуты + секунды + дата)

4. **Уровень честности**
   - Индикатор с прогресс-баром
   - Числовое значение
   - Описание (высокий/средний/низкий)

5. **Детализация ответов** (если доступна)
   - Все вопросы теста
   - Ответы пользователя
   - Правильность каждого ответа
   - Полученные баллы

## Цветовая схема

### Статус теста
- ✅ **Сдан**: зеленый (`#16a34a`)
- ❌ **Не сдан**: красный (`#dc2626`)

### Уровень честности
- ✅ **Высокий** (0-14): зеленый (`#16a34a`)
- ⚡ **Средний** (15-39): оранжевый (`#d97706`)
- ⚠️ **Низкий** (40+): красный (`#dc2626`)

## Fallback режим

Если библиотека `dompdf` не установлена:
- PDF отображается как HTML страница
- Добавлена кнопка "🖨️ Сохранить как PDF"
- Пользователь может сохранить через диалог печати браузера

## Рекомендуемая установка dompdf

Для production рекомендуется установить библиотеку dompdf:

```bash
cd C:\xampp\htdocs\test-platform
composer init
composer require dompdf/dompdf
```

После установки PDF будет генерироваться автоматически без fallback режима.

## Примеры использования

### Администратор
1. Войти как админ
2. Открыть `/admin.php`
3. Вкладка "📊 Результаты"
4. Кнопка "📄 Экспорт PDF"
5. PDF скачивается автоматически

### Студент (из истории)
1. Войти в личный кабинет `/dashboard.php`
2. Вкладка "📋 История"
3. Найти нужный результат
4. Кнопка "📄 PDF" в колонке "Действия"
5. PDF откроется в новой вкладке

### Студент (после теста)
1. Пройти тест
2. На экране результатов
3. Кнопка "📄 Скачать PDF"
4. PDF откроется в новой вкладке
